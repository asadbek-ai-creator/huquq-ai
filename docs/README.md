# huquqAI Documentation

This directory contains documentation for the huquqAI legal knowledge base system.

## Documentation Files

- `api.md` - API documentation
- `ontology.md` - Ontology structure documentation
- `usage.md` - Usage guide
- `development.md` - Development guide
- `karakalpak_guide.md` - Karakalpak language guide

## Building Documentation

To build the documentation using Sphinx:

```bash
cd docs
sphinx-build -b html . _build
```

## Contributing

When adding new features, please update the relevant documentation files.

## Language Support

All documentation should be available in:
- Karakalpak (kaa)
- English (en)
- Uzbek (uz)
- Russian (ru)
